# Portfolio

Este repositório contém o código-fonte do meu portfólio pessoal, uma aplicação **SPA (Single Page Application)** desenvolvida para apresentar meus projetos e experiências.

🌐 Acesse o projeto em: [portfolio.luffes.dev](https://portfolio.luffes.dev)

## Tecnologias

O projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS
- React
- Vite

--- 

## Como rodar o projeto?

Se deseja rodar o projeto localmente, basta seguir os seguintes passos:

1. Baixar as dependências dentro da pasta principal do projeto.

```
npm install
```

2. Iniciar o servidor local do projeto

```
npm run dev
```
